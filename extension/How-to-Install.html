LEADSNIPER 3.0
STEP 01
Unzip the Package
Extract the downloaded zip archive (LeadSniper_Secure.zip). You will see a folder named LeadSniper and this instruction guide. Keep the LeadSniper folder in a safe directory where you won't accidentally delete or move it.
解压安装包
将下载的 LeadSniper_Secure.zip 压缩包解压。解压后会获得一个名为 LeadSniper 的文件夹（包含插件的核心代码）和此指南文件。请将该文件夹移动到一个不易被误删的本地目录。

STEP 03
Enable Developer Mode
Locate the Developer Mode switch in the top-right corner of the extensions page and turn it ON.
🛡️ Security Verification: LeadSniper is loaded directly via the browser's developer tools. All your API keys and webhook settings are stored 100% locally on your machine. No scripts run externally.
开启开发者模式
在扩展程序管理页面的右上角，找到【开发者模式】的开关并将其开启。
🛡️ 安全验证： LeadSniper 是通过官方开发者模式本地加载运行的。您所有的 API Key 和 Webhook 数据都 100% 保存在本地，没有任何外部收集脚本，确保数据隐私绝对安全。
STEP 04
Load Unpacked Extension
1. Click the Load unpacked button in the top-left corner.
2. Choose and select the LeadSniper folder that you extracted in Step 1.
3. Once loaded, click the Puzzle icon in the browser toolbar, find LeadSniper, and click the Pin icon to pin it for easy access.
✅ SYSTEM SECURED & ACTIVE
加载已解压的扩展程序
1. 点击管理页面左上角的【加载已解压的扩展程序】（或 Load unpacked）按钮。
2. 在弹出的文件夹选择框中，选择在步骤 1 解压出来的 LeadSniper 文件夹。
3. 加载完成后，点击浏览器右上角的【拼图】图标，找到 LeadSniper，点击【固定】(Pin) 图标将其锁定到工具栏。
✅ 系统加载就绪，雷达已启动
